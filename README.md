# Javones_Mazmorras-G10
Sistema de personajes RPG Javones y Mazmorras G10
Este proyecto consiste en crear una especie de JUEGO RPG, mediante un codigo JAVA, que se base en diferentes clases, todo llamandolo desde un MAIN, para que quede mas limpio.

Ponerle todos los personajes, PERSONAJE, MAGO, GUERRERO, LADRON.

Añadiendo un INVENTARIO y diferentes EQUIPAMIENTOS.
